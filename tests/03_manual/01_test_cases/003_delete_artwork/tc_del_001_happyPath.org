** Test Objective
Verify that the user is able to delete an artwork from the table
** Pre-conditions
- Application is running
- The table has at least one row
** Test Steps
1. Click on an existing row
2. Press <delete>
3. Type the title in the search bar
4. Check if the row doesn't reappear
** Expected Results
- The selected row is imidietely deleted
- Other rows are not affected
- Picture thumbnail is deleted from the folder
- Confirmation message is being displayed
- The picture doesn't reappear in search filter
