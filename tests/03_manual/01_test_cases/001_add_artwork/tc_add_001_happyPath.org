** Test Objective
- Verify that a user can successfully add a valid artwork to the database, including images and metadata
** Pre-conditions
- Application is running
- Database is accessible (empty or populated)
** Test Steps
1. Navigate to the form section
2. Fill all the fields with valid data:
   - author: Emily Turner
   - title: Silent Horizon
   - size: 90x90cm
   - medium: Acrylic on Canvas
   - year: 2025
3. Click "Browse" and select JPG or PNG image  
4. Click "Add an Artwork"
** Expected Results
1. Artwork is saved successfully
2. Table updates immediately with the new artwork
3. Thumbnail displays correctly
4. No existing rows are affected
5. Proper success message appears
** Notes / Edge Cases
- Test with empty database
- Test with sorting on/off
